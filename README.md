🚀 Transmaa – Backend System for Logistics Platform

This repository contains the backend implementation of Transmaa, a logistics and truck booking platform developed during my 3-month internship at Edufyi 🎓.

💻 As the Backend Developer, I designed and developed the core server-side logic that powers the system.

⚙️ Backend Responsibilities:
🔹 Built RESTful APIs using Node.js and Express.js
🔹 Integrated MongoDB for database management 🗄️
🔹 Implemented Customer Booking API for load booking 📦
🔹 Developed Driver Registration & Login APIs 👨‍✈️
🔹 Created Load Listing & Accept Load APIs 🚚
🔹 Implemented Delivery Status Update APIs 📍
🔹 Developed Admin Approval & Order Status APIs 🛠️
🔹 Structured backend architecture for smooth frontend integration 🔗

✨ This project strengthened my practical skills in API development, database integration, and backend architecture.

🛠️ Tech Stack: Node.js ⚡ | Express.js 🌐 | MongoDB 🗄️ | JavaScript 💻
